Dimmerly Support

Control your external display brightness from the menu bar.

Troubleshooting

Displays don't dim

  • Ensure your external display supports DDC/CI brightness control.
  • Some USB-C hubs or docks may not pass through DDC commands — try connecting the display directly to your Mac.
  • If using the App Store version, screens are blanked rather than put to sleep. Move your mouse or press any key to restore.

Keyboard shortcut doesn't work

  1. Open System Settings > Privacy & Security > Accessibility.
  2. Make sure Dimmerly is listed and enabled.
  3. Restart Dimmerly after granting permission.
  4. If the shortcut still doesn't work, try changing it in Settings > Shortcuts to avoid conflicts with other apps.

Brightness slider has no effect

  • Not all displays support software brightness control. Built-in MacBook displays are not supported — Dimmerly controls external displays only.
  • Verify your monitor supports DDC/CI (most modern monitors do).

Menu bar icon not visible

  • If your menu bar is crowded, macOS may hide some icons. Try removing other menu bar items or using a tool like Bartender.
  • Restart Dimmerly from Applications.

Frequently Asked Questions

Does Dimmerly work with built-in displays?

Dimmerly is designed for external displays. Your Mac's built-in display brightness is controlled through System Settings or the keyboard brightness keys.

How many presets can I save?

You can save up to 10 brightness presets. Each preset can have its own keyboard shortcut for quick access.

Does Dimmerly collect any data?

No. Dimmerly does not collect, store, or transmit your data, and it does not use analytics or tracking. The app does not send data to developer-operated servers. All settings are stored locally on your Mac. If you choose "Use Current Location" for sunrise/sunset schedules, macOS Location Services may use Apple-provided location infrastructure. See our Privacy Policy.

What's the difference between the App Store and source versions?

The App Store version runs in Apple's sandbox and dims/screens blanks displays using gamma tables. The source version can put displays to sleep using macOS system commands and includes DDC/CI hardware monitor control features (when supported by your display/connection). Feature differences are documented in the README and release notes.
