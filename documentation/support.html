Dimmerly Support

Per-display brightness, warmth, presets, and external monitor controls from the menu bar.

Troubleshooting

Displays don’t dim

  • If using the App Store version, Dimmerly uses software dimming and screen blanking. Move your mouse, press a key, or press Escape depending on your wake setting.
  • Some displays and display modes do not respond equally to software gamma changes. Try changing brightness, warmth, or contrast separately to confirm which controls affect your display.
  • If using the direct download or source version with DDC/CI enabled, ensure your external display supports DDC/CI brightness control and try connecting the display directly to your Mac.

Keyboard shortcut doesn’t work

  1. Open System Settings › Privacy & Security › Accessibility.
  2. Make sure Dimmerly is listed and enabled.
  3. Restart Dimmerly after granting permission.
  4. If the shortcut still doesn’t work, try changing it in Settings › Shortcuts to avoid conflicts with other apps.

Brightness slider has no effect

  • The App Store version uses software brightness through gamma tables. Some displays, HDR modes, or color profiles may limit the visible effect.
  • The direct download or source version can use DDC/CI hardware brightness for supported external monitors. Some USB-C hubs, docks, DisplayLink adapters, TVs, or monitor settings may block DDC/CI.

Menu bar icon not visible

  • If your menu bar is crowded, macOS may hide some icons. Try removing other menu bar items or using a tool like Bartender.
  • Restart Dimmerly from Applications.

Frequently Asked Questions

Does Dimmerly work with built-in displays?

Dimmerly shows connected displays, including the built-in display where supported. The App Store version uses software dimming. The direct download or source version may use additional hardware brightness controls where macOS and the display support them.

How many presets can I save?

You can save up to 10 presets. Each preset can have its own keyboard shortcut for quick access.

Does Dimmerly collect any data?

No. Dimmerly does not collect or transmit personal data, and it does not use analytics or tracking. The app does not send data to developer-operated servers. Settings are stored locally on your Mac. If you choose “Use Current Location” for sunrise/sunset schedules, macOS Location Services may use Apple-provided location infrastructure. See our Privacy Policy.

What’s the difference between the App Store and source versions?

The App Store version runs in Apple’s sandbox and dims and screen-blanks displays using gamma tables. The source version can put displays to sleep using macOS system commands and includes DDC/CI hardware monitor control features when supported by your display and connection. Feature differences are documented in the README and release notes.

Contact

For bug reports and feature requests, please open an issue on GitHub Issues.

When reporting a bug, please include:

  • Your macOS version
  • Dimmerly version (from Settings › About)
  • Your display model and connection type
  • Steps to reproduce the issue
