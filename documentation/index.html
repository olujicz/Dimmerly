Dimmerly for macOS

Display controls that belong in the menu bar.

Dimmerly gives each screen its own brightness, warmth, contrast, presets, schedules, and sleep controls—without digging through monitor menus.

Download for macOS Requires macOS 15 Sequoia or later

Install the full build with Homebrew.

brew tap olujicz/dimmerly
brew install --cask dimmerly

Built for multi-display Macs.

Direct controls for the parts of a display setup that macOS and monitor buttons leave scattered.

01

Control each display separately

Adjust brightness, warmth, and contrast without changing every screen at once.

02

Use the monitor’s hardware controls

DDC/CI adds brightness, contrast, volume, mute, and input switching where the display supports them.

03

Save complete display setups

Recall presets instead of rebuilding the same brightness and warmth settings by hand.

04

Automate the routine

Use schedules, global shortcuts, desktop widgets, and Shortcuts actions.

Local settings. Public source.

Dimmerly has no analytics, tracking, ads, or app-managed network requests. It is open source under the MIT License.
